# 🏆 TFS - Tournament For Small Servers

Bienvenue sur le projet officiel de **TFS (Tournament For Small Servers)**.

TFS est une plateforme qui organise des tournois entre petits serveurs Discord afin de permettre aux communautés de se rencontrer, de gagner en visibilité et de s'affronter dans une compétition basée sur les votes.

---

## 🚀 Objectif

Créer un espace où les petits serveurs Discord peuvent :

- 🏆 Participer à des tournois
- 📈 Gagner en visibilité
- 🤝 Découvrir d'autres communautés
- 🎁 Remporter des récompenses

---

## ⚔️ Fonctionnement

1. Les serveurs Discord s'inscrivent au tournoi.
2. Deux serveurs s'affrontent.
3. Les membres votent pour leur serveur préféré.
4. Le gagnant passe au tour suivant.
5. Le dernier serveur restant remporte le tournoi.
